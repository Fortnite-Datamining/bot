# bot

Invite the bot [here](https://discord.com/oauth2/authorize?client_id=1483047779411431518) and run `/setup`

## Commands 

- `/setup` - set the channel for datamining updates 
- `/remove` - stop posting updates in a channel
- `/status` - show which channels are receiving updates
- `/notify` - get dm'd when a skin appears in the item shop
- `/unnotify` - remove a skin from your wishlist
- `/wishlist` - view your wishlist
